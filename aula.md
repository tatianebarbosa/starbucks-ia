# Anotações de aula

## Ferramentas usadas

- [x] VS Code
- [x] Live Server
- [x] GitHub Theme ou Dracula
- [x] Material Icon Theme
- [x] Portuguese (Brazil)
- [x] Color Highlight

## Conteúdos praticados

- [x] HTML para montar a estrutura da página
- [x] CSS para estilizar layout, cores e responsividade
- [x] JavaScript para trocar bebidas, personalizar pedido e controlar carrinho
- [x] LocalStorage para salvar o carrinho no navegador

## HTML

- [x] Tags principais: `html`, `head`, `body`, `header`, `main`, `section`, `footer`
- [x] Atributos importantes: `class`, `id`, `src`, `alt`, `href`, `aria-label`
- [x] Formulários com `input`, `textarea`, `fieldset`, `legend` e `button`

## CSS

- [x] Seletores por tag, classe e estado
- [x] Variáveis de cor com `:root`
- [x] Layout com `grid` e `flex`
- [x] Responsividade com `@media`
- [x] Estados visuais com `:hover`, `:focus-visible` e classes dinâmicas

## JavaScript

- [x] Variáveis e constantes
- [x] Funções
- [x] Eventos de clique, mudança e envio de formulário
- [x] Manipulação do DOM
- [x] Cálculo de preço e atualização do carrinho
